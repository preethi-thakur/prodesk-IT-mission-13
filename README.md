Designed and developed TaskMatrix, a Jira-inspired task management application featuring a Kanban board, task CRUD operations, drag-and-drop functionality, Redux state management, dark mode, responsive design, and persistent local storage using Next.js and Tailwind CSS.


# TaskMatrix

TaskMatrix is a modern project management application inspired by Kanban workflows. It helps teams organize, track, and manage tasks through an intuitive drag-and-drop interface with a clean and responsive user experience.

## Features

- Kanban board with drag-and-drop functionality
- Create, edit, duplicate, and delete tasks
- Task search and filtering
- Dark and light theme support
- Responsive design for desktop and mobile
- Task details drawer
- Dashboard with project metrics
- Local storage persistence
- Redux Toolkit state management
- Smooth animations with Framer Motion

## Tech Stack

- Next.js
- React
- JavaScript
- Redux Toolkit
- Tailwind CSS
